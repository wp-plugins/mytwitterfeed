== Description ==

MyTwitterFeed is a simple and easy to user plugin for WordPress which shows the latest tweets of your Twitter account. 

Easy embed tweets in one of your sidebars with an easy and very simple to use WordPress widget.

== Installation ==

1. Download the MyTwitterFeed WordPress plugin, unzip it and put it in the directory wp-content/plugins/ of your website. 
2. Activate the plugin through the 'Plugins' menu in WordPress

Configure:

- Name of twitter account to display tweets from
- Number of tweets to show

== Frequently Asked Questions ==

= What does mean the #hastags and @mentions configuration =

You can choose if you want to show the hashtags and/or mentions in your tweet
